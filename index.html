Alchemy Game

For my A-level Computer Science coursework, I decided to create an Alchemy game where users can select elements to see how they bond in a reaction chamber.
To achieve this each element is trying to fill their outer shell and looking to share electrons. When they find an element to share with both elements then attract each other.
I used an object-oriented approach in python to calculate the movement of the selected elements and the pygame library for the GUI.
